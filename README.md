# Swibrary Privacy Policy Site

Static GitHub Pages site for the Swibrary privacy policy.

## Files

- `index.html` — privacy policy page
- `styles.css` — warm bookshelf-style design
- `assets/swibrary_icon.png` — app icon / favicon

## Before publishing

Replace every `YOUR_CONTACT_EMAIL` placeholder in `index.html` with the contact email used in Google Play Console.

## GitHub Pages

Upload these files to a GitHub repository, then enable GitHub Pages from the repository settings.
